Quote Results Page Demo

This demo shows the new quote results component. It calculates pricing based on distance and displays interactive maps with route information.

Features Included:

  • ✅ Distance-based pricing calculation
  • ✅ Interactive Google Maps with route display
  • ✅ Multiple transport options (shuttle/private)
  • ✅ Zone-based pricing overrides
  • ✅ Responsive design
  • ✅ URL parameter parsing for pickup/dropoff

Note: This demo will try to load pickup and dropoff locations from URL parameters, or use defaults if none are provided.
