Quote Request Form Demo

This demo shows the new quote request form component. It includes all the features from the original booking form but is now clearly separated as a quote request system.

Features Included:

  • ✅ Google Maps Places autocomplete
  • ✅ Date and time picker validation
  • ✅ Passenger count selection
  • ✅ Field validation and error handling
  • ✅ Loading states and visual feedback
  • ✅ Shared CSS foundation
