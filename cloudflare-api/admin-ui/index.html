API Documentation

All admin endpoints require an Authorization header with your API key.

Client Management

GET /admin/clients
List all clients
POST /admin/clients
Create a new client
PUT /admin/clients/:id
Update a client
DELETE /admin/clients/:id
Delete a client

Tour Management

GET /admin/tours?client_id={id}
List all tours (filter by client)
POST /admin/tours
Create a new tour
PUT /admin/tours/:id
Update a tour
DELETE /admin/tours/:id
Delete a tour
